## Linear Models

When we say that $ y $ is a linear function of $x$, we mean that the graph of the function is a
line, so we can use the slope-intercept form of the equation of a line to write a formula for the function as

$$ y = f(x) = mx + b $$

$$ P(x) = x^3 + x^2 + b + c$$
